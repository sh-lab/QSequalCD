# プロジェクト仕様定義（更新版・確定）

本プロジェクトはカードゲーム「QS=CD」を実装する。

## 思想・前提

- QS = CD
- Q = 1
- よって S = CD
- C ≥ 1
- D ≥ 1
- Q（品質）は固定値1であり、操作不可能
- Qは数値として扱わない
- 品質は交渉対象ではなく前提条件である
- プレイヤーが管理する対象は S（スコープ）である
- C（コスト）と D（デリバリー）は、S を実現するために必要な制約である
- 1 の S を実現するためには、それに見合った C と D が必要である
- C や D を十分に確保しないまま S を追求する行為は、本来成立しない要求を押し付けることを意味する
- その結果として発生する負荷や疲弊を、本ゲームでは Q の低下として捉える
- たとえ成果物の品質が維持されていたとしても、関係者の QOL が損なわれている状態は望ましくない

## ゲーム概要

- 基本は1人用カードゲーム
- プレイヤーはPM（プロジェクトマネージャー）
- 目的はスコアとコストの両立である
- 最終スコアが目標スコア以上であること
- 最終コストがコスト制約以下であること

## 変数

- S（スコープ）
  - 成果量を表す
  - ゲーム上はスコアとして集計する
- C（コスト）＝場に存在するメンバーカード数
  - ただし、手札カードはコストに含めない
  - サビ残強制を行った追加ラウンドで配られたメンバーカードは、例外的に最終コストに含めない
- D（デリバリー）＝ラウンド数
  - 基本3ラウンド
  - 最大5ラウンド
- Q＝1
  - 固定値
  - 数値としては扱わない

## 初期設定

- チーム人数：3〜6
- 一人あたりのスコア見込み：3〜6
  - このスコア見込みは全メンバー共通の値である
  - ゲームを通して固定され、個別メンバーごとに変化しない
- 目標スコア
 - デフォルト：63
- コスト制約
 - デフォルト：30

## ラウンド構造

- 1ラウンド＝1行
- 横方向はメンバー列
- 列＝メンバー
- 行は下から上へ時間順に積み重なる
- 各行の列数はチーム人数分で固定する
- ただし、離脱済みメンバー列にはカードを配らず、空白として扱う

## メンバーカード

- 合計35枚

### 数値カード

数値カードは0〜6で構成される。

配分：

- 0：2枚
- 1：3枚
- 2：4枚
- 3：6枚
- 4：10枚
- 5：7枚
- 6：2枚

### 特殊カード

- メンバー離脱：1枚

## メンバーカードの配布

- 各ラウンド開始時、チーム人数分の列を確認する
- 離脱していないメンバー列にのみ、山札からメンバーカードを1枚ずつ配る
- 配ったカードはすべて裏向きで、そのラウンドの行に横一列で配置する
- 離脱済みメンバー列にはカードを配らず、空白として扱う

## 表にする枚数のルール

- 各ラウンドにおいて、必ず「表にする枚数」までメンバーカードを表にする
- 表にするカードは、裏向きで配られたカードの中から任意に選ぶ
- 表にしないという選択は存在しない
- 表にする枚数は、算出された上限枚数と、そのラウンドで実際に配られたメンバーカード枚数の小さい方とする

## 表にする枚数の上限

- 一人あたりのスコア見込みにより、基本上限を決定する
  - 見込み3：1枚
  - 見込み4：2枚
  - 見込み5：3枚
  - 見込み6：4枚
- チーム人数が6人の場合、上限をさらに+1枚する
- 追加ラウンドでは、上記の上限にさらに+1枚する
- 表にする枚数が増えることは、人数増加や無茶によってプロジェクトの不確実性が増していることを表す

## メンバーカードの処理

ラウンド終了時、場に存在するメンバーカードを処理する。

- 表向きカード：
  - カード記載の数値、またはカード効果をスコアとして扱う
- 裏向きカード：
  - 初期に決めた全メンバー共通のスコア見込みをスコアとして扱う
- 離脱済みメンバー列：
  - カードは存在しない
  - コスト0
  - スコア0

## 離脱カード

- 離脱カードが表向きで出た場合、次ラウンド以降、該当メンバー列にはカードを配らない
- 離脱後の該当メンバー列は、次ラウンド以降：
  - コスト0
  - スコア0
- 離脱カードは、手札②（無効化）によってそのラウンドのみ無効化できる
- 無効化された離脱カードは、そのラウンドでは裏向きカードとして扱う
- 無効化された場合、離脱効果は発生しない

## 手札

- 各ゲーム開始時、手札は5枚で開始する
- ゲーム中（ラウンド間）に手札補充は行わない
- 継続ゲーム開始時には、手札は再び5枚に補充される
- 各ラウンド開始時、メンバーカードを表にする前に：
  - そのラウンドで使用する手札の枚数（0〜2枚）
  - 各手札の使用方法
  を事前に決定する
- 1ラウンドに使用できる手札は最大2枚
- 使用した手札は、使用したラウンドの場に残る
- 手札カードはメンバーカードではないため、コストには含めない

### 手札の使用方法

#### ① 3点スコアとして使用

- 手札1枚を、3点スコアとしてそのラウンドの場札の右側に出す
- コストには含めない
- プレイングマネージャーとして直接成果を出す行為を表す

#### ② 無効化として使用

- 手札1枚を使い、表向きにしたメンバーカード1枚の効果を無効化する
- 無効化されたカードは、そのラウンドでは裏向きカードとして扱う
- スコア計算時は、初期に決めた全メンバー共通のスコア見込みを使用する
- マネージャーとしてトラブルを予防する行為を表す
- 手札②を使用する場合、使用すること自体はメンバーカードを表にする前に宣言する
- 対象となる表向きメンバーカードは、カードを表にした後で選択する

## 追加ラウンド

- 3ラウンド終了時に目標未達の場合のみ追加可能
- 最大2ラウンド
- 追加ラウンドを行うことで、総ラウンド数は最大5ラウンドとなる
- 追加ラウンドでは、表にする枚数の上限にさらに+1枚する
- 追加ラウンドにおいて、本来発生するコストを支払わない行為を「サビ残強制」と定義する
- サビ残強制を行った追加ラウンドでは、そのラウンドで新たに配られたメンバーカードを最終コストに含めない
- サビ残強制そのものでは即敗北しない
- サビ残強制を行った結果として、ゲーム終了時に継続カード「監査」が出た場合、該当ゲームはスコアに関係なく即敗北とする

### ゲーム終了時の集計順

1. ラウンド結果から基本スコアを計算する
2. 既に保持している継続カードの補正を適用する
3. 最終スコアを確定する
4. 勝利条件を判定する
5. 継続カードを引く

## 勝利条件

以下を両方満たした場合に勝利する。

- 最終スコアが目標スコア以上であること
- 最終コストが指定された上限値以下であること

## 継続ゲーム

- ゲーム終了時、各メンバーごとに継続カードを1枚引く
- ただし、離脱済みメンバーは継続カードを引かない
- 継続カード効果は累積する
- 次ゲームの目標スコアは前回ゲームの最終スコア以上とする
- 継続ゲーム開始前に、PMはメンバーの入れ替えおよびチーム人数の変更を行うことができる
- メンバーの入れ替え、追加、離任など、チーム構成に変更があった場合、チームスコアアップは失効する

## 継続カード

継続カードは合計14枚で構成される。

### 監査：1枚

- サビ残強制という禁断の行為を行ったPMへの事後的な罰
- サビ残強制を行っていた場合、該当ゲームはスコアに関係なく即敗北
- 場に残らない

### 離任：1枚

- 対象メンバーが次ゲーム以降のチームから外れる
- 場に残らない

### チームスコアアップ：2枚

- 最終スコアにチーム人数分を加算する
- メンバーの入れ替え、追加、離任など、チーム構成に変更があった場合は失効する

### メンバーコストアップ：3枚

- 対象メンバーの最終コストに+1する
- 継続ゲームで場に残る

### メンバースコアアップ：3枚

- 対象メンバーの最終スコアに+2する
- 継続ゲームで場に残る

### コスト削減圧力：1枚

- 上位組織からのコスト削減要求を表す
- 次ゲーム以降のコスト制約を1減らす
- 場に残らない

### 何もなし：3枚

- 効果なし
- 場に残らない

### 継続カード山札の再構築

- 各ゲーム終了時、場に残らない継続カードを残りの継続カードと合わせてシャッフルし、新しい継続カードの山札とする
- 離任やチーム変更によって失効した継続カードも、場に残らない継続カードとして扱う

## 継続カードの配置

- メンバー系カードは、1行目の下、対応するメンバーの列に配置する
- メンバー系カードとは、以下を指す
  - メンバーコストアップ
  - メンバースコアアップ
- チーム系カードは専用の別領域に配置する
- チーム系カードとは、以下を指す
  - チームスコアアップ
